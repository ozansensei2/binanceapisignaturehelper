Binance API — Signed Request Example

Import into Postman

  1. Click Download ZIP below. Unzip it and use the .json file inside (do not import the ZIP itself into Postman).
  2. In Postman: Import → choose that JSON file.
  3. Select the collection → Variables: set apiKey and apiSecret (the X-MBX-APIKEY header uses {{apiKey}}).
  4. If you exported while Auth type was HMAC-SHA256 (signed), the Pre-request script adds timestamp and the HMAC signature before send — click Send.
  5. For unsigned modes (None or API key in header only) there is no signature script; ensure your keys match the environment (mainnet / testnet).
